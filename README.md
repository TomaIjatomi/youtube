## How this repo is organised

Each episode lives in its own folder under `episodes/`. Folders are numbered in publish order so they stay in sequence regardless of topic.

Inside each episode folder you'll find:
- A `README.md` explaining what the code does and how to run it
- The actual demo code referenced in the video
- Any data, prompts, or configs needed to reproduce the results

---

## Getting started

Each episode has its own setup instructions in its README. In general:

```bash
# Clone the repo
git clone https://github.com/tomaijatomi/youtube.git
cd youtube

# Navigate to the episode you're interested in
cd episodes/002-five-defence-patterns

# Follow that episode's README from there
```
